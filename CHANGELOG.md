# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added

- Initial release of Vulcan Neo component library
- **Atoms**: Surface, Button, InsetField, Switch, Checkbox, Radio, RadioGroup, Badge, Avatar, IconButton, FeaturedIcon, Slider, ProgressBar, Tag, Tooltip, RatingStars, SocialButton, AppStoreButton, ButtonGroup, Textarea, VerificationCodeInput, BadgeGroup
- **Molecules**: Card, Select, Dropdown, Tabs, Modal, SlideoutMenu, Alert, Notification, Calendar, DatePicker, Breadcrumbs, Pagination
- **Organisms**: Table, Metrics, ActivityFeed, CodeSnippet, FileUploader, Carousel, CommandMenu, HeaderNavigation, SidebarNavigation, Messaging, ProgressSteps, LoadingIndicator, EmptyState, InlineCTA
- **Charts**: LineChart, BarChart, PieChart (powered by D3.js)
- **Marketing Sections**: MarketingHeader, HeroSection, FeaturesSection, PricingSection, CTASection, StatsSection, NewsletterSection, TestimonialSection, LogoCloudSection, BlogSection, ContactSection, TeamSection, CareersSection, FAQSection, ContentSection, Footer, Banner
- **Templates**: ErrorPage (404/500/403 pages)
- **Foundation Documentation**: Colors, Typography, Motion
- Design tokens with CSS variables
- Light and dark theme support
- TypeScript definitions
- Storybook documentation
- GitHub Actions CI/CD pipeline
- GitHub Pages deployment for Storybook
